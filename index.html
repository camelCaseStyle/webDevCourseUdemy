Lets learn EMV!

This site offers tools for calculating the TVR, TSI, TAC and other EMV goodies. All this while, it also aims to help you learn more about how EMV works.

TVR

TVR stands for transaction verification result. The TVR is a series of bits set by the terminal reading an EMV card, based on logical tests (for example has the card expired).

TAC

TSI stands for transaction action code. The terminal has to decides either to proceed the transaction offline, to go online or to reject the transaction.

TSI

TSI stands for transaction status indicator. Indicates the functions performed in a transaction

CDOL

CDOL stands for Card Risk Management Data Object List
